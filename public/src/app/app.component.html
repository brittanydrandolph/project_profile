 Products & Categories Built with C#, .NET Core, Entity and MySQL

In this web application, the user can add various products and categories to the database. Then, using the power of Entity and MySQL, the user can create Many To Many relationships between the two groups where many products can belong to many categories, and many categories can be home to many products! For example, the products bacon and eggs can both belong to the categories "Food" and "Breakfast items".

Register for Classes Built with C#, .Net Core, Entity and MySQL

In this web application, users can login to thier account to register for classes! In order to register for a class, the user must have enough money in their bank account to cover the cost. To ensure this, users can add money to their account by clicking the "View Your Bank Account" link! After adding the desired amount, the user's Dashboard Action options will change to "Join" or remain at "Add money to join" depending on how much money was deposited. Additionally, users can add a class of thier own they would like to teach or edit the class if needed. Lastly, the user can click on the class title to view details about the class and view who has registered for the class. If the user changes thier mind, they can deregister from the Dashboard by clicking the "Leave" Action option.
